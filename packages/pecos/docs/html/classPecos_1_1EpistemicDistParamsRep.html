Detailed Description

The representation of a set of epistemic distribution parameters. This representation, or body, may be shared by multiple EpistemicDistParams handle instances.

The EpistemicDistParams/EpistemicDistParamsRep pairs utilize a handle-body idiom (Coplien, Advanced C++).
