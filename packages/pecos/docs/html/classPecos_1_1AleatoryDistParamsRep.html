Detailed Description

The representation of a set of aleatory distribution parameters. This representation, or body, may be shared by multiple AleatoryDistParams handle instances.

The AleatoryDistParams/AleatoryDistParamsRep pairs utilize a handle-body idiom (Coplien, Advanced C++).
