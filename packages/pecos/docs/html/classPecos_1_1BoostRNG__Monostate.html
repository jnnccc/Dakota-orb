Detailed Description

The monostate BoostRNG class provides static pointers randomNum and randomNum2 that at runtime will either point to BoostRNG_Monostate::mt19937, or be overridden to point to defaultnum1 / defaultrnum2. At construct time they point to the BoostRNG_Monostate generators.
