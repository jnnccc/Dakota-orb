Calculate one of several different types of error norms of a set of vectors.

Each column of the reference_values matrix is considered independently with the correpsonding column in the approximation_values. The error between the each set of two columns tested is returned.

Parameters
reference_valuesa matrix containing the reference values
approximate_valuesa matrix containig the approximate values
active_columnsspecifies which set of columns to consider. If empty all columns are considered normalise if true the error norms are normalised. The l_inf norm is normalised by the half the range of the data in the reference_values column. The L_2 norm is normalised by the standard deviation of the data in the reference values column.
